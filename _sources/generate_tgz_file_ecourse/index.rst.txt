Generate the tgz file of Ecourse
================================

From this page you can learn how to generate the file build.tgz of Ecourse

Execute the build file of Ecourse
#################################

First, clone the git repository of "Ecourse" and run the build process:

.. code-block:: none

  $ cd Inria-Chile/tranque
  $ git clone git@gitlab.com:Inria-Chile/tranque/ecourse.git
  $ cd ecourse
  $ ./build.sh

A build.tgz should be generated into the ecourse folder.
