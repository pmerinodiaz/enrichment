Generate the tgz file of Emac
=============================

From this page you can learn how to generate the file build.tgz of EMAC.

Execute the build file of EMAC
#################################

First, clone the git repository of "EMAC" and run the build process:

.. code-block:: none

  $ cd Inria-Chile/tranque
  $ git clone git@gitlab.com:Inria-Chile/tranque/emac.git
  $ cd emac
  $ ./build.sh

A build.tgz should be generated into the emac folder.
