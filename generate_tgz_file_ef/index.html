Generate the tgz file of EF¶

From this page you can learn how to generate the file build.tgz of EF.

Execute the build file of EF¶

First, clone the git repository of “Estabilidad física” and run the build process:

$ cd Inria-Chile/tranque
$ git clone git@gitlab.com:Inria-Chile/tranque/ef.git
$ cd ef
$ ./build.sh

A build.tgz should be generated into the ef folder.
